Rules used by this app
  • The number of loan officers can be any size.
  • Each loan is assigned to one random officer.
  • The loans themselves are shuffled before assignment.
  • Assignments are then distributed in random fair rounds, so everyone gets one before anyone gets a second, everyone gets two before anyone gets a third, and so on.
